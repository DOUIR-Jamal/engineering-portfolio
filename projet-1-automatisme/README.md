# Système de Tri Automatisé
### Commandé par API Schneider Electric Zelio Logic SR3

## Description
Ce projet porte sur la conception, la programmation et la simulation d'un système de tri automatisé capable de classer des objets selon leur taille. Réalisé dans le cadre du Mini-projet en 2ème année Génie Mécatronique à l'ENSA Safi.

## Objectif
Concevoir, programmer et simuler un système de tri automatisé basé sur un cahier des charges industriel, en intégrant les principes de l'automatisme séquentiel.

## Technologies utilisées
- GRAFCET 
- Langage Ladder (API Schneider Electric Zelio Logic SR3)
- Zelio Soft 2
- FluidSIM (simulation pneumatique)
- Maquette virtuelle 2D

## Contenu
- Rapport technique (PDF)
- Présentation PowerPoint (PPTX)
- Programme Ladder (Zelio Soft 2)
- Schémas GRAFCET et captures de simulation
- Photos de la maquette réelle

## Compétences développées
- Conception de systèmes automatisés
- Programmation d'automates industriels (API)
- Simulation de systèmes pneumatiques
- Analyse de cahier des charges
